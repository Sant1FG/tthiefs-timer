# Tthief's timer

Módulo de extensión para FoundryVTT que amplía las mecánicas de combate mediante un temporizador.

##  Descripción

Plugin desarrollado sobre la API de FoundryVTT que registra el tiempo real transcurrido desde el inicio de la iniciativa hasta el final del combate, facilitando la monitorización de encuentros.

##  Tecnologías

- **Lenguaje:** JavaScript
- **Plataforma:** FoundryVTT (API)

##  Características

- Temporización automática desde el inicio de la iniciativa
- Visualización del tiempo transcurrido al finalizar el combate
- Compatible con la API oficial de FoundryVTT

##  Instalación

1. Accede a la última release disponible en GitHub
2. Copia la URL del atributo `manifest` dentro de `module.json`
3. Pega la URL en la pantalla de instalación de módulos de FoundryVTT
4. Activa el módulo desde la configuración de tu mundo
